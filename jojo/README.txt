/////////////////////////// COMPILATION ///////////////////////////

Pour compiler et éxécuter le programme il suffit de lancer la commande "make" dans le Terminal.
Un fichier crypto sera créer,il est possible de relancer le programme avec la commande "./crypto"
dans le Terminal si le fichier "crypto.c" a déjà été compilé

/////////////////////////// Execution Du Programme ///////////////////////////

Lorsque vous allez lancer le programme ,plusieurs tests s'afficheront dont celui du PGCD et de Jacobi
ainsi que celui du Programme entier.

Pour le Programme:

Vous aurez alors la possibilité de choisir quel nombre tester parmi la liste que vous nous avez
fourni.Il faudra entrer un numéro compris entre 1 et 10 selon votre choix.
Ensuite vous aurez la possibilité de choisir le nombre d'itérations pour la fonction de Solovay-Strassen.
Pour finir le résultat du Programme s'affichera.

Choix 11:

Si vous entrez le choix 11,le programme vous demandera le nombre d'itérations et ensuite il vous proposera
de choisir un  nombre entier.(celui que vous voulez)
Il executera le programme et renverra le résultat.
