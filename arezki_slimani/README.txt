Ce Programme vous permet de tester si un nombre est premier ou composé.

Pour pouvoir l'utiliser, vous devez etre sur un environnement linux.
Les differentes etapes de compilation et de teste sont :
	-Pour compiler le programme vous avez simplement besoin d'ecrire make sur votre terminal.
	-un menu vous permettra de choisir de selectionner les liste de nombre deja present, ou bien d'entrer un nombre de votre choix

1- Vous avez selectionner la liste de nombre :
	Après selection vous aurez encore 10 choix de nombre premier deja existant a tester, Tapez le nombre associé a chacun pour le tester.
	Par la suite vous pourrez entrer le nombre d'iteration que vous voulez pour votre teste.

2- Vous voulez entrer comme ca vous chante :
	Vous aurez tout d'abord a entrer le nombre de votre choix, puis le nombre d'itteration (le nombre de fois que le nombre est tester avec differente combinaisant.).

Apres avoir fait vos choix, le resultat s'affiche directement sur le terminal.

pour faire d'autre teste vous devez relancer l'application avec un make , et pour effacer les fichier executable vous devez faire un 'make clean'.

Voila maintenant vous maitrisez a la perfection mon programme !!!.
